# My Omarchy Setup

## Included Configurations
- Neovim (with plugins)
- Shell configs (bash/zsh)
- Git configuration
- Postman settings

## Installation
```bash
git clone https://github.com/yourusername/dotfiles.git ~/dotfiles
cd ~/dotfiles
./bootstrap.sh
```

## Manual Steps After Install
- Import Postman collections from `postman/`
- Configure any API keys
